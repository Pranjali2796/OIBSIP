# OIBSIP

This repository contains **five Python-based projects** developed during the  
**Oasis Infobyte Python Development Internship**.

The projects demonstrate the application of core Python concepts such as automation, API integration, and networking.

---

## Technologies Used
- Python 3  
- Socket Programming  
- Multithreading  
- REST API Integration  
- Speech Recognition and Text-to-Speech  

---

## Projects

### Task 1: Voice Assistant
A voice-controlled Python assistant that responds to spoken user commands.

- Speech recognition and text-to-speech  
- Provides current date and time  
- Performs Google searches  

---

### Task 2: BMI Calculator
A console-based application to calculate Body Mass Index (BMI).

- Accepts height and weight as input  
- Calculates BMI  
- Displays health category  

---

### Task 3: Random Password Generator
Generates secure random passwords based on user preferences.

- Custom password length  
- Letters, numbers, and symbols options  

---

### Task 4: Weather Application
Fetches real-time weather data using an external API.

- City-based weather search  
- Displays temperature, humidity, and condition  

---

- Client-server architecture
